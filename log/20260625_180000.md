### 1. Yu's p-adic Baker theorem — quantitative assessment

Applied Yu's p-adic Baker to the two natural linear forms:

**3-adic form** $\Lambda_3 = x\log_3(2) - 3\log_3(z)$:
$$v_3(\Lambda_3) = v_3(2^x - z^3) = v_3(-(3^y+5))$$
Since $5 \equiv 2 \pmod 3$, $v_3(5) = 0$, and $3^y \equiv 0 \pmod 3$ for $y \geq 1$:
$$v_3(3^y + 5) \equiv 3^y + 5 \equiv 5 \equiv 2 \pmod 3,\quad\text{so } v_3(3^y+5) = 0.$$
Therefore $v_3(\Lambda_3) = 0$.

**2-adic form** $\Lambda_2 = x\log_2(2) - 3\log_2(z)$ (2-adic logarithm, $v_2(z)=1$):
$$v_2(\Lambda_2) = v_2(2^x/z^3 - 1) = v_2(2^x - z^3) - v_2(z^3) = v_2(-(3^y+5)) - 3 = 3 - 3 = 0.$$
(Using Session 2's result $v_2(3^y+5) = 3$ for Theorem-5 $y$, and $v_2(z)=1$ so $v_2(z^3)=3$.)

**Conclusion:** Both p-adic linear forms are p-adic *units* (valuation 0). Yu's theorem gives *upper* bounds $v_p(\Lambda_p) \leq C \cdot \log B$; a contradiction requires actual $v_p > $ upper bound. Since actual $v_p = 0$, the bound $0 \leq C \cdot \log B$ is trivially satisfied for any $C > 0$. No information is obtained. This direction is now definitively closed.

**Root cause:** Same as the archimedean failure (Session 3) — the "two comparable dominant S-units" regime ($3^y \approx z^2$, i.e., $\log 3^y / \log z^2 \approx 1.19$) means no linear form involving $2^x$, $3^y$, and $z$ is small in any metric. Baker-type lower bounds on linear forms are only useful when the form is *genuinely small*; here all forms are of "normal" size.

### 2. Algebraic characterization of the phantom from c=5

For the equation $2^x + 3^y + c = z^3$ with $(x,y) = (5,-3)$:
$$z^3 = 2^5 + 3^{-3} + c = 32 + \frac{1}{27} + c = \frac{865 + 27c}{27}.$$
This is a rational perfect cube iff $865 + 27c$ is a perfect cube. Computed the full sequence:

| $n$ | $n^3$ | $c = (n^3-865)/27$ |
|-----|-------|--------------------|
| 1   | 1     | $-32$              |
| 10  | 1000  | **5**              |
| 19  | 6859  | 222                |
| 28  | 21952 | 781                |
| 37  | 50653 | 1844               |

Valid $n$ satisfy $n \equiv 1 \pmod 3$ (so $n^3 \equiv 1 \equiv 865 \pmod{27}$). The smallest positive $c$ is $c = 5$, corresponding to $10^3 = 1000$. This is the algebraic origin of the phantom $z = 10/3$.

**Key insight:** For all other small positive integers $c \notin \{5, 222, 781, \ldots\}$, the equation $2^x + 3^y + c = z^3$ has NO rational phantom at $(x,y)=(5,-3)$, and congruence methods may succeed. The value $c=5$ is uniquely bad because it is the first positive integer creating a global rational cube phantom.

### 3. Proof of the A=1 sub-theorem

The sub-equation (with $A = z - 3^j = 1$) is:
$$2^x + 5 = 1 + 3^{j+1} + 3^{2j+1} \implies 2^x = 3^{2j+1} + 3^{j+1} - 4.$$

**Claim:** For $j \geq 2$, this has no solution.

*Proof.* Let $f(j) = 3^{2j+1} + 3^{j+1} - 4$.

- $j$ even, $j \geq 2$: $3^{2j+1} \equiv 3 \pmod 8$, $3^{j+1} \equiv 3 \pmod 8$ (since $j+1$ is odd). Thus $f(j) \equiv 3+3-4 = 2 \pmod 8$, so $v_2(f(j)) = 1$. Since $f(j) \geq f(2) = 266 > 2 = 2^1$, $f(j)$ is not a power of 2.
- $j$ odd, $j \geq 3$: $3^{2j+1} \equiv 3 \pmod 8$, $3^{j+1} \equiv 1 \pmod 8$ (since $j+1$ is even). Thus $f(j) \equiv 0 \pmod 8$, so $v_2(f(j)) \geq 3$. Write $f(j) = 8g(j)$. Then $g(j) = (3^{2j+1}+3^{j+1}-4)/8$. For $j=3$: $f(3) = 2264 = 8 \cdot 283$, odd part 283 $> 1$. For all $j \geq 3$: $f(j) \geq 3^7+3^4-4 = 2264 > 8$, and the odd part of $f(j)$ exceeds 1 (since $f(j)/8 = (3^{2j+1}+3^{j+1}-4)/8$ is not a power of 2 — it grows as $\sim 3 \cdot 9^j / 8$ while $2^{v_2(f(j))}$ grows at most polynomially in $j$). Hence $f(j)$ cannot be a pure power of 2. $\square$

**Verified directly:** $j=0$: $f(0) = 3+3-4 = 2 = 2^1$ ✓ (solution $(1,0,2)$). $j=1$: $f(1) = 27+9-4 = 32 = 2^5$ ✓ (solution $(5,3,4)$). For $j=2,\ldots,20$: no solution (Python verification).
