## Work Done

### 1. Correcting the gpf($z^3 - 5$) reformulation

The directions file claimed: "if gpf($z^3-5$) grows effectively with $z$, only finitely
many $z$ can have $z^3-5$ a $\{2,3\}$-integer, which is exactly the Kuromine condition."

**This is a conflation.** A $\{2,3\}$-integer is an integer whose only prime factors are
2 and 3 (i.e., $n = 2^a 3^b$). The Kuromine condition is $z^3 - 5 = 2^x + 3^y$, which
does NOT require $z^3 - 5$ to be $\{2,3\}$-smooth.

**Wolfram verification:** $z = 4$ gives $z^3 - 5 = 59$ (prime, NOT $\{2,3\}$-smooth), yet
$59 = 32 + 27 = 2^5 + 3^3$, which IS a Kuromine solution ($x=5, y=3, z=4$).

**The correct reformulation:** The Kuromine condition requires $z^3 - 5$ to be
*representable as a sum $2^x + 3^y$* — a combinatorial condition on the additive
structure of $z^3 - 5$, not on its prime factorization. These are entirely different.

**Consequence for the gpf angle:** gpf($z^3 - 5$) growing with $z$ does NOT imply
$z^3 - 5$ cannot be written as $2^x + 3^y$. The direction as stated is a dead end
(the conflation is fundamental, not fixable). It is hereby retired.

### 3. Discovery of Theorem M (new, Session 14)

**Setup:** Any Theorem-5 solution has $y = 3j$ (Theorem 4) and the factorization
$z^3 - w^3 = 2^x + 5$ with $w = 3^j$, $A = z - w \geq 1$ (since $z^3 > w^3$).

**Key observation:** The minimum possible value of $z^3 - w^3$ for $z = w + 1$ is:
$$(w+1)^3 - w^3 = 3w^2 + 3w + 1 = 3^{2j+1} + 3^{j+1} + 1.$$

**Theorem M (new, Session 14):**
> Let $x, y \geq 0$ be integers with $y = 3j$. If $3^{2j+1} + 3^{j+1} + 1 > 2^x + 5$
> (equivalently, $3^{2j+1} > 2^x + 4$), then there is NO integer $z$ satisfying
> $z^3 = 2^x + 3^y + 5$.

**Proof:** Let $w = 3^j$. We have $w^3 < 2^x + 3^y + 5$ (since $2^x + 5 > 0$). If the
hypothesis holds, then $(w+1)^3 = w^3 + 3^{2j+1} + 3^{j+1} + 1 > w^3 + (2^x+5) = 2^x + 3^y + 5$.
So $w^3 < 2^x + 3^y + 5 < (w+1)^3$, and $2^x + 3^y + 5$ lies strictly between two
consecutive perfect cubes. Hence it is not a perfect cube. $\square$

**Verification on known solutions:**
- $(x,y) = (5,3)$: $j=1$, $w=3$. $3^3 + 3^2 + 1 = 37 = 2^5 + 5$. Hypothesis fails
  ($37 \not> 37$). Theorem M correctly does NOT rule out this solution.
- $(x,y) = (1,0)$: $j=0$, $w=1$. $3 + 3 + 1 = 7 > 2^1 + 5 = 7$? The hypothesis is
  $7 > 7$, which is FALSE. Theorem M correctly does not rule out this solution.

**Application to the Theorem-5 family:** For $x = 5 + 332640 k_x$, $j = 55439 + 55440 k_y$:
$$\text{Condition: } (2j+1)\log 3 > x \log 2 + \log(1 + 5/2^x) \approx x\log 2.$$
$$\text{Simplifying: } k_y > \frac{x \cdot \log 2 / \log 3 - 110879}{110880} \approx \frac{3\log 2}{\log 3} k_x - 1.$$

**Corollary for $k_x = 1$ (Session 14's strongest result):** If $x = 332645$ and $y =
166317 + 166320 k_y$ with $k_y \geq 1$, then $2^x + 3^y + 5$ is not a perfect cube.

*Proof:* For $k_y = 1$: $2j+1 = 221759$, $3^{221759} = 2^{351480} > 2^{332645} = 2^x$,
so $3^{2j+1} > 2^x > 2^x + 4$. By Theorem M, no solution exists. For $k_y \geq 2$, the
left side grows while $2^x$ is fixed, so the condition is even more strongly satisfied. $\square$

**General structure:** The surviving pairs $(k_x, k_y)$ satisfy $k_y/k_x <
3\log_3 2 - 1 \approx 0.893$. Wolfram computed: among $100 \times 201 = 20100$ pairs with
$k_x \in [1,100]$ and $k_y \in [0,200]$, exactly 9507 survive (53\% are ruled out).

### 4. Structure of the surviving cases

For the surviving cases ($k_y < $ threshold), $2^x \geq 3^{2j+1}$, which means $2^x$
is the dominant term: $z^3 = 2^x + 3^y + 5 \approx 2^x$, so $z \approx 2^{x/3}$.

The size of $A = z - w$: since $z \approx 2^{x/3}$ and $w = 3^j$, and the surviving
condition requires $2^{x/3} \geq 3^j = w$, we have $A \approx z \gg w$ in the
deep-interior of the surviving region, and $A \approx 1$ at the boundary.

The **boundary of Theorem M** ($2^x \approx 3^{2j+1}$, i.e., $A \approx 1$) is exactly
the regime studied in Session 5 (both known solutions have $A = 1$). The known solutions
live at the boundary. Any hypothetical Theorem-5 solution with $k_y \ll k_x$ would have
$A \gg 1$ and live in the interior — the hard regime for Baker-type methods.

### 5. Structural relationship to the two-S-units barrier

The critical ratio $k_y/k_x = 3\log_3 2 - 1$ comes from $x\log 2 = (2j+1)\log 3$, i.e.,
$2^x = 3^{y+\varepsilon} \approx 3^y$. This is EXACTLY the "two-comparable-S-units"
boundary where $2^x \approx 3^y$ — the regime that defeats Baker-type methods (Session 3).

- If $k_y/k_x > 3\log_3 2 - 1$ (i.e., $3^y \gg 2^x$): **Theorem M eliminates** the case.
- If $k_y/k_x \leq 3\log_3 2 - 1$ (i.e., $2^x \geq 3^y$ approximately): **Theorem M
  does not apply**; these are exactly the cases where Baker-type and all known analytic
  methods are also blocked (by the two-S-units barrier).

So Theorem M cleanly partitions the Theorem-5 parameter space at the two-S-units boundary,
eliminating the half where $3^y$ dominates. But the surviving half — where $2^x$
dominates — remains exactly as hard as before.

## Conclusion

Session 14 proves Theorem M, a new elementary result that rigorously eliminates the
"$3^y$ dominates" half of the Theorem-5 parameter space. This is the first provably
correct new theorem on the equation itself since Theorem 5 was established (Session 1).

The theorem is elementary and complete: no Baker bounds, no p-adic analysis, no modular
forms — just the observation that consecutive cubes are spaced at least $3w^2$ apart.

However, Theorem M is not sufficient to prove the conjecture: the surviving parameter
space (where $2^x \geq 3^y$) remains infinite and hard. The key unresolved case for
$k_x = 1$ is exactly the single pair $(k_x, k_y) = (1, 0)$, i.e., the question of
whether $2^{332645} + 3^{166317} + 5$ is a perfect cube. (It almost certainly is not,
but this specific case also resists Theorem M and all existing tools.)

The two-comparable-S-units barrier remains the core obstruction: the surviving cases
are exactly those where the existing analytic machinery fails.
