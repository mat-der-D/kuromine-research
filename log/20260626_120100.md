## §2. FRAMEWORK BLUEPRINTING (Sub-step 2)

**What kind of machinery is needed?**

The prior Session-20 framework coupled depth (3-adic) to size (archimedean) at a *single*
$k_x$. It failed because the coupling required an effective irrationality measure.

The new idea: exploit **inter-term constraints** — the structure of the map
$k_x \mapsto A(k_x)$ *between* consecutive integer arguments. This is a different type of
structure that does NOT immediately require an irrationality measure; it requires only:

(i) The 3-adic Taylor expansion of the analytic function $A(k_x)$, and
(ii) A constraint that integer-valued points on a 3-adic analytic curve must satisfy
     relative to the curve's own "derivative" data.

The appropriate machinery is a **3-adic analytic geometry of the solution curve**. Concretely:

- Define the "solution curve" $\mathcal{C} \subset \mathbb{Z}_3 \times \mathbb{Z}_3$ by
  $\mathcal{C} = \{(k_x, A) : A^3 = 32 g^{k_x} + 5\}$ — a 3-adic analytic curve.
- A genuine Kuromine solution with $k_x = k_0$ is a point $(k_0, A_0) \in \mathcal{C}$
  with $k_0 \in \mathbb{Z}_{>0}$ and $A_0 \in \mathbb{Z}_{>0}$ (integer coordinates).
- The curve $\mathcal{C}$ is smooth (non-singular) at every point with $A \not\equiv 0
  \pmod 3$, since $\partial(A^3)/\partial A = 3A^2 \not\equiv 0 \pmod 3$ iff $A \not\equiv
  0 \pmod 3$, which holds (Theorem I: $A \equiv 22 \not\equiv 0 \pmod 3$).
- The 3-adic implicit function theorem guarantees $A$ is a 3-adic analytic function of
  $k_x$ near any point on $\mathcal{C}$ — exactly the bijection established in Session 10.

**The new theoretical approach:** Instead of studying 3-adic depth vs archimedean size
at a single point, study the **3-adic geometry of integer points on $\mathcal{C}$** via
a "$p$-adic Chabauty-type" argument, but applied to the *exponential parametrization*
rather than an algebraic curve. This is a DIFFERENT direction from Session 13's (retired)
Chabauty-Kim: here the curve $\mathcal{C}$ IS algebraic in the $(k_x, A)$ variables over
$\mathbb{Z}_3$ (it is the zero set of $F(k, A) = A^3 - 32g^k - 5$ where $g^k =
\exp_3(k \log_3 g)$ is a 3-adic exponential). The "integer points" are those with $k_x \in
\mathbb{Z}_{>0}$. The constraint is not an algebraic constraint but an **integrality
constraint on the exponential parameter $k_x$**.

**Why this is different from Session 13's Chabauty-Kim:** Session 13's obstruction was
"$x$ and $y$ are exponents of fixed bases, not algebraic coordinates." In the new
framework, we have *already* passed through Theorem 5 and identified the parameter space
as $(k_x, k_y) \in \mathbb{Z}_{>0}^2$ with $k_x$ being a genuine algebraic input to a
3-adic analytic function. The question is not "is $(x, y)$ an integer" but "is $k_x$ an
integer for which $A(k_x)$ is also an integer." This is a 3-adic analytic problem in
the $(k_x, A)$ plane.

**Proposed mechanism:** The 3-adic analytic function $A: \mathbb{Z}_3 \to \mathbb{Z}_3$
is a bijection; its integer values at integer inputs are dense in $\mathbb{Z}_3$ (integers
are dense in $\mathbb{Z}_3$). So $A$ takes integer values at some integer $k_x$. However,
those integer values might be forced to grow at a specific 3-adic rate, and if that rate
is FASTER than the archimedean size $\approx 2^{x/3}$ allows, a contradiction arises.

**Key formula for the differential constraint:**

$$A(k_x + 1) - A(k_x) = A'(k_x) + O(A''(k_x)/2)$$

where (computing via the chain rule and implicit differentiation of $A^3 = 32 g^{k_x} + 5$):

$$3A^2 \frac{dA}{dk_x} = 32 g^{k_x} \log_3 g$$

$$\frac{dA}{dk_x} = \frac{32 g^{k_x} \log_3 g}{3 A^2} = \frac{(A^3 - 5) \log_3 g}{3 A^2}.$$

Since $v_3(A) = 0$, $v_3(A^2) = 0$, $v_3(\log_3 g) = 4$ (as $g \equiv 1 \pmod{3^4}$,

$$v_3\!\left(\frac{dA}{dk_x}\right) = v_3(32 g^{k_x}) + v_3(\log_3 g) - v_3(3) - v_3(A^2) = 0 + 4 - 1 - 0 = 3.$$

So **$dA/dk_x \in 3^3 \mathbb{Z}_3$**, i.e., $A'(k_x) \equiv 0 \pmod{27}$ for all
$k_x \in \mathbb{Z}_3$.

## §3. AXIOMATIC DEFINITION (Sub-step 3)

**Definition 1 (The Solution Curve).** The *Kuromine curve* over $\mathbb{Z}_3$ is the set
$$\mathcal{K} = \{(k, a) \in \mathbb{Z}_3 \times \mathbb{Z}_3 : a^3 = 32 \exp_3(k L) + 5, \quad a \equiv 1 \pmod 3\},$$
where $L = \log_3(g) = \log_3(2^{332640}) \in 3^4 \mathbb{Z}_3 \setminus 3^5 \mathbb{Z}_3$
and $\exp_3, \log_3$ are the 3-adic exponential and logarithm (convergent on $3\mathbb{Z}_3$
and $1 + 3\mathbb{Z}_3$ respectively, extended to all of $\mathbb{Z}_3$ via the structure
of $g$).

**Definition 2 (The Integer Solution Locus).** $\mathcal{K}_{\mathbb{Z}} = \{(k, a) \in
\mathcal{K} : k \in \mathbb{Z}_{>0},\, a \in \mathbb{Z}_{>0}\}$. A Theorem-5 solution
with parameter $k_x$ corresponds to a point $(k_x, A(k_x)) \in \mathcal{K}_{\mathbb{Z}}$.

**Definition 3 (The 3-adic Tangent Speed).** For $k_0 \in \mathbb{Z}_3$ and the implicit
function $A(k)$ defined by $\mathcal{K}$ near $k_0$, the *3-adic tangent speed* is
$$\tau(k_0) = \frac{dA}{dk}\bigg|_{k=k_0} = \frac{(A(k_0)^3 - 5) L}{3 A(k_0)^2} \in 3^3 \mathbb{Z}_3.$$
We have $v_3(\tau(k_0)) = 3$ universally (for all $k_0$, since $v_3(A(k_0)^3 - 5) = v_3(32 g^{k_0})
= 0$, $v_3(L) = 4$, $v_3(3) = 1$, $v_3(A^2) = 0$).

**Definition 4 (The Differential Depth).** For integers $k_0 < k_1$ in $\mathbb{Z}_{>0}$,
define the *inter-term 3-adic depth*
$$\Delta(k_0, k_1) = v_3(A(k_1) - A(k_0)).$$
By the mean-value-type estimate for 3-adic analytic functions (Mahler's theorem / Strassmann),
$$\Delta(k_0, k_1) \geq v_3(\tau(k_0)) + v_3(k_1 - k_0) = 3 + v_3(k_1 - k_0).$$
(This follows because $A(k)$ is a 3-adic power series in $k - k_0$ with leading term
$\tau(k_0)(k-k_0)$ and all higher-order terms at least as 3-adically divisible as
$\tau(k_0)^2 / \text{const}$.)

**Definition 5 (Integer-Distance Constraint).** If two integers $k_0 < k_1$ both give
integer values $A(k_0), A(k_1) \in \mathbb{Z}_{>0}$, then $A(k_1) - A(k_0) \in \mathbb{Z}$.
Combining with Definition 4:
$$v_3(A(k_1) - A(k_0)) \geq 3 + v_3(k_1 - k_0) \geq 3 \quad \text{(for any two distinct integer } k).$$
So any two integer-argument integer-output values of $A$ differ by a multiple of $27$.

*Note:* This is a non-trivial constraint: it says $A(k_0) \equiv A(k_1) \pmod{27}$ for any
two integer parameters giving integer outputs. This is AUTOMATICALLY SATISFIED by Theorem I:
all branches give $A \equiv 22 \pmod{27}$ universally. So this weaker form of the
differential depth constraint is already recorded and is non-contradictory.

### Testing: Does the Differential Depth give anything new?

**Lemma C (Consecutive-Integer Difference).** For any two consecutive integer inputs
$k_x$ and $k_x + 1$,
$$A(k_x + 1) - A(k_x) \equiv \tau(k_x) \pmod{3^4 \mathbb{Z}_3}.$$
More precisely, using the Taylor expansion $A(k+1) = A(k) + A'(k) + A''(k)/2 + \ldots$
and $v_3(A''(k)) \geq ?$:

The second derivative:
$$\frac{d^2A}{dk^2} = \frac{d}{dk}\!\left[\frac{(A^3-5)L}{3A^2}\right]
= \frac{3A^2 A' \cdot L \cdot 3A^2 - (A^3-5)L\cdot 6A A'}{(3A^2)^2} \cdot L$$

This simplifies (using $A' = (A^3-5)L/(3A^2)$) to:
$$A'' = A' \cdot L \cdot \frac{3A^2 \cdot 3A^2 - 6A(A^3-5)}{9A^4}
= A' L \cdot \frac{9A^4 - 6A^4 + 30A}{9A^4}
= A' L \cdot \frac{3A^4 + 30A}{9A^4}.$$

Wait — let me redo this more carefully. We have $F(k,A) = A^3 - 32g^k - 5 = 0$. Implicit
differentiation:

$$\frac{dA}{dk} = -\frac{F_k}{F_A} = -\frac{-32 g^k L}{3A^2} = \frac{32 g^k L}{3A^2}.$$

Second derivative (implicit differentiation of $A' = 32 g^k L/(3A^2)$ with respect to $k$):

$$A'' = \frac{32 g^k L^2}{3A^2} - \frac{32 g^k L \cdot 6A A'}{9A^4}
= \frac{32 g^k L^2}{3A^2} - \frac{2 (32 g^k L)^2}{9 A^5}.$$

Now $v_3(32 g^k L^2/(3A^2)) = 0 + 4 \cdot 2 - 1 - 0 = 7$. And
$v_3((32 g^k L)^2/(9A^5)) = 2 \cdot 4 - 2 - 0 = 6$.

So $v_3(A'') = 6$ (the second term dominates in 3-adic absolute value):
$$A'' \equiv -\frac{2(32 g^k L)^2}{9A^5} \pmod{3^7 \mathbb{Z}_3}, \quad v_3(A'') = 6.$$

**Conclusion from Taylor expansion:**
$$A(k+1) = A(k) + A'(k) + \frac{A''(k)}{2} + O(3^9)$$
with $v_3(A') = 3$, $v_3(A''/2) = 6 - 0 = 6$ (since $2$ is a 3-adic unit), so
$$A(k+1) - A(k) \equiv A'(k) \pmod{3^6}, \quad v_3(A(k+1) - A(k)) = 3.$$

This is Lemma C confirmed analytically: **the 3-adic distance between consecutive integer
inputs is exactly $3^3$ (i.e., $v_3 = 3$)**. This is consistent with what we know: $A$
takes values in $22 + 27\mathbb{Z}_3$ (Theorem I / J), so integer values differ by
multiples of 27 = $3^3$. No contradiction.

### Testing: Is the archimedean magnitude of $A(k_x+1) - A(k_x)$ constrained?

For genuine integers $A(k_x), A(k_x+1)$ (if solutions existed at both $k_x$ and $k_x+1$),

$$|A(k_x+1) - A(k_x)| \approx \left|\frac{32 g^{k_x} L_\infty}{3 A(k_x)^2}\right|_\infty$$

$$|A(k_x+1) - A(k_x)| \approx \frac{2^{5+332640 k_x} \cdot 332640 \log 2}{3 \cdot 2^{2(5+332640 k_x)/3}}
= \frac{332640 \log 2}{3} \cdot 2^{(5+332640 k_x)/3} \approx \frac{332640 \log 2}{3} \cdot A(k_x).$$

**This gives NO contradiction by itself** — it simply says that consecutive hypothetical
