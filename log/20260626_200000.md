## Work Done

### 1. Formalisation of the Case II window (the "from-above" argument)

**Setup.** For a Theorem-5 solution write $z = 3^j + A$ with $A = z - 3^j \geq 1$.
Session 15 proved (Wolfram) that no surviving pair ever has $A \lesssim 10^{24}$; thus
$A$ is always large. Two regions arise naturally:

- **Case I:** $A \geq 3^j$ (i.e., $z \geq 2 \cdot 3^j$). No upper bound on $A$ from this
  direction alone.
- **Case II:** $0 < A < 3^j$ (i.e., $3^j < z < 2 \cdot 3^j$). The "narrow band."

Sub-question (b) focuses on Case II. Since $B = A^2 + 3^{j+1}A + 3^{2j+1}$:

$$A < 3^j \Rightarrow B = A^2 + 3^{j+1}A + 3^{2j+1} < (3^j)^2 + 3^{j+1}\cdot 3^j + 3^{2j+1}
= 3^{2j} + 3^{2j+1} + 3^{2j+1} = 3^{2j}(1 + 2\cdot3) = 7\cdot3^{2j}.$$

$$A \geq 1 \Rightarrow B > 3^{2j+1}.$$

So in Case II: $3^{2j+1} < B < 7 \cdot 3^{2j}$.

**The Case II window for $A$:** Since $AB = 2^x + 5$ and $B \in (3^{2j+1}, 7 \cdot 3^{2j})$:

$$\frac{2^x + 5}{7 \cdot 3^{2j}} < A < \frac{2^x + 5}{3^{2j+1}}.$$

Setting $D = x\log 2 - (2j+1)\log 3$ (the "Theorem-M parameter"), $3^{2j+1} \approx 2^x e^{-D}$,
so the window becomes approximately:

$$\frac{e^D}{7} < A < \frac{e^D}{3}.$$

**Width of the window:** $e^D/3 - e^D/7 = 4e^D/21 \approx 0.19\, e^D$.

**Consequence:** For any surviving Theorem-5 pair $(k_x, k_y)$ with $D > 0$ (which all
surviving pairs have, by Theorem M), the window contains approximately $0.19\,e^D / 432$
integers satisfying $A \equiv 427 \pmod{432}$ (Theorem K). For $k_x = 1$, $k_y = 0$:
$D \approx 8757$ and the window contains $\sim e^{8757}/2268 \approx 10^{3799}$ compatible
integers. No shortage of candidates.

### 2. Norm form structure of $B$ and the "$p \equiv 2 \pmod 3$" constraint

$B = z^2 + z \cdot 3^j + 3^{2j} = N_{\mathbb{Z}[\omega]}(z - 3^j \omega)$ is an Eisenstein norm.
Since $\mathbb{Z}[\omega]$ is a PID and primes $p \equiv 2 \pmod 3$ are inert in $\mathbb{Z}[\omega]$,
any such prime divides $B$ to an **even power**. Since $\gcd(A,B) = 1$, this means:

> **Every prime $p \equiv 2 \pmod 3$ that divides $2^x + 5 = A \cdot B$ to an odd power
> must divide $A$** (not $B$).

This is a genuine structural constraint on the factorization. However, by the phantom
obstruction, for every prime $p \neq 3$ there is a "phantom assignment" $A \equiv 3$,
$B \equiv 37/3 \pmod p$ that satisfies all local conditions. The constraint above is
automatically compatible with the phantom (since $3 \not\equiv 0 \pmod p$ for $p \neq 3$,
the phantom survives). So this norm-form divisibility constraint adds no new information
beyond what's already captured by the soft obstruction.

### 3. 2-adic analysis of $f(A) = A^3 + 3^{j+1}A^2 + 3^{2j+1}A - 5$

For $A < 3^j$ (Case II), the constraint $2^x = f(A)$ must produce an exact power of 2.
Using $A \equiv 11 \pmod{16}$ (Theorem E/K) and $j \equiv 7 \pmod 8$:

| quantity | residue | derivation |
|---|---|---|
| $A^3 \bmod 16$ | $11^3 = 1331 \equiv 3$ | direct |
| $3^{j+1} \bmod 16$ | $\equiv 1$ | $\text{ord}_{16}(3)=4$, $j+1 \equiv 0 \pmod 4$ |
| $3^{j+1}A^2 \bmod 16$ | $1 \cdot 121 \equiv 9$ | $11^2 = 121$ |
| $3^{2j+1} \bmod 16$ | $\equiv 11$ | $\text{ord}_{16}(3)=4$, $2j+1 \equiv 3 \pmod 4$ |
| $3^{2j+1}A \bmod 16$ | $11 \cdot 11 = 121 \equiv 9$ | |
| $5 \bmod 16$ | 5 | |

Sum: $f(A) \equiv 3 + 9 + 9 - 5 = 16 \equiv 0 \pmod{16}$. So $v_2(f(A)) \geq 4$.

This is consistent with $v_2(2^x) = x \gg 4$ but adds no new information — the 2-adic
analysis is non-obstructive by design (the cube map is a bijection on 2-adic units, as
proved in Session 2).

### 4. Why sub-question (b) cannot yield a contradiction

The "narrow-band" argument (Case II window) provides:
- A window $A \in (e^D/7, e^D/3)$ — exponentially wide (width $\sim 0.19e^D$).
- The window always contains $\sim e^D/2268$ integers with the right congruences.
- The 3-adic bijection $k_x \mapsto A$ (Theorem J) maps every $k_x \in \mathbb{Z}_3$ to
  some $A \in \mathbb{Z}_3$; since $\mathbb{Z}$ is dense in $\mathbb{Z}_3$, no 3-adic
  obstruction can prevent an integer $A$ from lying in the window.
- The 2-adic analysis is non-obstructive (Session 2: cube map is 2-adic bijection).
- The norm-form divisibility constraint is satisfied by the phantom.

**Diagnosis:** Sub-question (b) encounters the same triple obstruction as every other
approach: (i) phantom obstruction for all $p \neq 3$; (ii) "37 is a cube mod $3^n$ for
all $n$" wall for $p = 3$; (iii) 2-adic cube bijection for $p = 2$. The "narrow-band"
reformulation does not escape any of these three obstructions. It provides a useful
SIZE WINDOW for $A$, but this window is too large (exponential width) to yield a
contradiction from any known technique.

**Sub-question (b) is hereby CLOSED negatively.**

### 6. A new structural observation: the Shorey-Tijdeman obstruction

The Shorey–Tijdeman method (1976) for bounding solutions of $z^n = 2^x + c$ (single
exponential on RHS, all others fixed) uses Baker's method to bound $x$ in terms of
$n, z, c$. Applied to Kuromine with one exponential fixed (say, treating $3^y + 5$ as
a "constant" for fixed $y$), it would give a bound on $x$ in terms of $y$. But this
bound has the form $x \leq C(y)$ where $C(y)$ is enormous (Baker-type, depending
exponentially on $y$), which is entirely consistent with the Theorem-5 family's
$x \approx 2y$ relationship. No contradiction.

Applying it the other way (treating $2^x + 5$ as "constant" for fixed $x$): Baker's
method bounds $y$ in terms of $x$, again consistent. This recovers the known finiteness
per fixed $x$ (or $y$), which is the classical statement that the equation $z^3 - 3^y = c$
(for fixed $c$) has finitely many solutions. The Theorem-5 family requires uniformity
over $x$, which Baker cannot provide.

**Conclusion:** The Shorey–Tijdeman approach gives nothing new; it reduces to the same
effective Baker method already retired in Session 5.
