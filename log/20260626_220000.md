## Work Done

### 1. Function-field analog: the equation without the constant

**Setup.** Replace $2^x, 3^y, z$ by polynomials $P^x, Q^y, H$ over $\mathbb{C}[t]$,
where $P, Q$ are fixed irreducible polynomials and $H \in \mathbb{C}[t]$ is the unknown.
The equation without the constant reads:
$$P(t)^x + Q(t)^y = H(t)^3.$$

**Applying Mason-Stothers** to $P^x + Q^y - H^3 = 0$ (three pairwise coprime terms,
since the original equation has $\gcd(A,B)=1$ and $H^3 = P^x + Q^y$ has RHS a sum of
$P$-power and $Q$-power which are coprime):
$$\max(x\deg P,\; y\deg Q,\; 3\deg H) \leq \deg(\text{rad}(P^x\cdot Q^y \cdot H^3)) - 1 = \deg P + \deg Q + \deg(\text{rad}(H)) - 1.$$

Since $\deg(\text{rad}(H)) \leq \deg H$:
$$3\deg H \leq \deg P + \deg Q + \deg H - 1,$$
$$2\deg H \leq \deg P + \deg Q - 1.$$

For $\deg P = \deg Q = 1$: $\deg H \leq 0$, i.e., $H$ is a constant — **no non-constant
solution**. More generally, $\deg H \leq (\deg P + \deg Q - 1)/2$, which is finite and
independent of $x$ and $y$.

Using $3\deg H \approx x\deg P$ (when $P^x$ dominates): $x \leq 3\deg H / \deg P
\leq 3(\deg P + \deg Q - 1)/(2\deg P)$, which is bounded. So for fixed $P, Q$, only
finitely many values of $x$ (and $y$) admit a non-constant solution $H$.

**Conclusion:** The function-field Kuromine equation WITHOUT the additive constant is
completely resolved by Mason-Stothers — only finitely many non-constant solutions,
bounded in terms of $\deg P + \deg Q$.

### 2. Function-field analog: with the constant 5

With the constant, the equation is $P^x + Q^y + 5 = H^3$. Apply Mason-Stothers to
$(P^x + 5) + Q^y - H^3 = 0$:

$$\text{rad}((P^x + 5) \cdot Q^y \cdot H^3)$$

The crucial term is $\text{rad}(P^x + 5)$. Since $P = t - \alpha$ (say, linear) and
$P^x + 5 = (t-\alpha)^x + 5$ is a squarefree polynomial of degree $x$ over $\mathbb{C}$
(its derivative $x(t-\alpha)^{x-1}$ is coprime to $(t-\alpha)^x + 5$ since the latter
evaluated at $\alpha$ is $5 \neq 0$), we have:
$$\deg(\text{rad}(P^x + 5)) = x \deg P.$$

The radical is **full-size** (degree $x\deg P$), so Mason-Stothers gives:
$$3\deg H \leq \deg(\text{rad}((P^x+5) \cdot Q^y \cdot H^3)) - 1 \leq x\deg P + \deg Q + \deg H - 1,$$
$$(3-1)\deg H \leq x\deg P + \deg Q - 1,$$
which is NOT a contradiction since $3\deg H \approx x\deg P$ gives $2\deg H \approx \frac{2}{3}x\deg P < x\deg P + \deg Q - 1$ for any $\deg Q \geq 1$.

**Conclusion:** Mason-Stothers gives NO useful bound in the presence of the additive
constant 5. The same "phantom-like" obstruction appears: the constant $5$ introduces a
term $P^x + 5$ whose radical is full-size, preventing radical compression, which is
exactly what makes Mason-Stothers work in the no-constant case.

### 3. Structural theorem: the constant is the hard ingredient

**Theorem (Session 17, analytic):** Let $P, Q$ be irreducible polynomials over
$\mathbb{C}$ and $c \in \mathbb{C}^*$ a non-zero constant. Then:

(a) The equation $P^x + Q^y = H^3$ (no constant) has only finitely many solutions with
$H$ non-constant, bounded by $\deg P + \deg Q$.

(b) The equation $P^x + Q^y + c = H^3$ (with constant) is NOT resolved by
Mason-Stothers: the bound degenerates because $\text{rad}(P^x + c)$ has degree
$x\deg P$ (the polynomial $P^x + c$ is squarefree for $c \neq 0$).

**Proof.** Part (a): Mason-Stothers as above. Part (b): $P^x + c$ is squarefree over
$\mathbb{C}$ for any nonzero constant $c$ (the derivative $(P^x)' = xP^{x-1}P'$ shares
no root with $P^x + c$ since $P^x + c$ vanishes where $P = (-c)^{1/x}$, while $P^{x-1}$
vanishes where $P = 0$, different points for $c \neq 0$). Hence $\deg(\text{rad}(P^x+c))
= x\deg P$, and the Mason-Stothers bound becomes vacuous. $\square$

**Corollary:** In the function field, the difficulty introduced by adding the constant $c$
is provably real, not an artifact of method. Any proof of the function-field analog (even)
must use something beyond Mason-Stothers / polynomial-ABC.

### 4. The phantom obstruction as a function-field phenomenon

In the integer case, the phantom $(x,y,z) = (5,-3,10/3)$ is a rational solution that
"appears" modulo every prime $p \neq 3$, blocking all local methods. The function-field
obstruction is analogous: the additive constant creates a "virtual" solution where the
radical is not compressed.

**Explicit function-field phantom:** Over $\mathbb{C}(t)$ (rational functions), the
equation $P^x + Q^y + 5 = H^3$ has the "phantom" rational function solution:
$$P^x + 5 = R^3, \quad Q^y = 0 \text{ (trivial)},$$
i.e., if $R(t)^3 = P(t)^x + 5$ has a solution over $\mathbb{C}(t)$. For instance,
with $P = t$ and $x = 3$: $t^3 + 5$ is a polynomial, and $H = (t^3+5)^{1/3}$ is a
valid solution over $\overline{\mathbb{C}(t)}$ but not a polynomial. The "phantom-like"
rational function solutions prevent algebraic-function-field methods from ruling out
polynomial solutions.

The parallel is imperfect (integer phantom is a rational number, not an integer;
function-field phantom is a rational function, not a polynomial), but the structural
role is the same.

### 5. New structural observation: mod-37 analysis

For Theorem-5: $y = 166317 + 166320 k_y$. Since $166320 = 9240 \cdot 18$ exactly,
$y \equiv 166317 \pmod{18}$. And $166317 = 9240 \cdot 18 - 3$, so
$$y \equiv -3 \equiv 15 \pmod{18} \text{ for ALL Theorem-5 solutions (independent of } k_y\text{).}$$

Therefore $3^y \equiv 3^{15} = 3^{-3} \equiv 27^{-1} \equiv 11 \pmod{37}$ (since
$27 \cdot 11 = 297 = 8 \cdot 37 + 1 \equiv 1$).

So for all Theorem-5 solutions: $z^3 = 2^x + 3^y + 5 \equiv 0 + 11 + 0 = 11 \pmod{37}$.
Since $11^{12} \equiv 1 \pmod{37}$ (verified: $11^2 \equiv 10$, $11^4 \equiv 26$,
$11^8 \equiv 10$, $11^{12} = 11^8 \cdot 11^4 \equiv 10 \cdot 26 = 260 \equiv 1$), $11$
is a cubic residue mod 37. **No contradiction.** This is consistent (and expected from
the phantom compatibility).

This computation does yield the universal congruence $z^3 \equiv 11 \pmod{37}$ for all
Theorem-5 solutions, an explicit new constraint (though non-contradictory).

### 6. The algebraic origin of the $c = 5$ specialness: cyclotomic perspective

$c = 5$ is characterized by the identity $37 \cdot 27 + 1 = 1000 = 10^3$, equivalently
$37 \mid \Phi_3(10) = 10^2 + 10 + 1 = 111 = 3 \cdot 37$. Here $\Phi_3$ is the 3rd
cyclotomic polynomial. Primes dividing $\Phi_3(n)$ are either 3 or $\equiv 1 \pmod 3$;
$37 \equiv 1 \pmod 3$ and $37 \mid \Phi_3(10)$.

The identity $2^5 + 5 = 37 = \Phi_3(10)/3$ links the base $x = 5$, the constant $c = 5$,
and the phantom's numerator $z = 10$. Specifically:
- $2^5 + 5 = 37$: the value of $2^x + c$ at $x = 5$.
- $37 = \Phi_3(10)/3 = (10^2 + 10 + 1)/3$: the norm-form value of $z - w$ in the phantom.
- $10 = 2 \cdot 5$: the phantom's numerator $z$.

**This is a precise three-way resonance:** The identity $2^5 + 5 = (10^2+10+1)/3$ can
be read as: "the value of $2^x + c$ at $x = 5$ equals the cyclotomic factor
$\Phi_3(z)/\Phi_3(z)/3$ at $z = 10$." This is the algebraic origin of the phantom.

**Implication for other values of $c$:** For the phantom to exist, we need rational
$z, w > 0$ with $2^5 + c = z^3 - w^3 = (z-w)(z^2+zw+w^2)$ and $w = 3^{-1} = 1/3$.
This gives $32 + c = (z - 1/3)(z^2 + z/3 + 1/9)$. Multiplying by 27:
$27(32+c) = (3z-1)(9z^2+3z+1)$. Setting $u = 3z - 1$ (so $u > 0$, $z = (u+1)/3$):
$27(32+c) = u(u^2+3u+3) = u^3 + 3u^2 + 3u$.
For $u = 9$ (i.e., $z = 10/3$): $u^3 + 3u^2 + 3u = 729 + 243 + 27 = 999 = 27 \cdot 37$.
So $c = 37 - 32 = 5$. ✓

For other $u$: $u = 8$: $512 + 192 + 24 = 728 = 27 \cdot 26.96...$, not divisible by 27.
$u = 10$: $1000 + 300 + 30 = 1330$; $1330/27$ is not an integer.
$u = 6$: $216 + 108 + 18 = 342 = 27 \cdot 12.67$, not divisible by 27.
$u = 3k$: $27k^3 + 27k^2 + 9k$; for this to be $27(32+c)$ need $9k \equiv 0 \pmod{27}$,
i.e., $k \equiv 0 \pmod 3$, so $k = 3m$: $u = 9m$ giving $27(32+c) = 729m^3 + 243m^2 + 27m$,
i.e., $32 + c = 27m^3 + 9m^2 + m$.

For $m=1$ ($u=9$, $z=10/3$): $c = 27+9+1-32 = 5$. ✓ (the known case)
For $m=2$ ($u=18$, $z=19/3$): $c = 27\cdot8 + 9\cdot4 + 2 - 32 = 216 + 36 + 2 - 32 = 222$.
For $m=3$ ($u=27$, $z=28/3$): $c = 27\cdot27 + 9\cdot9 + 3 - 32 = 729 + 81 + 3 - 32 = 781$.

So: **the only small positive integer $c$ for which the Kuromine-type equation
$2^x + 3^y + c = z^3$ has a phantom solution with $y = -3$ is $c = 5$
(for $m=1$, $z = 10/3$).** Nearby phantoms require $c = 222$ or $c = 781$, which are
much larger constants. This makes $c = 5$ genuinely special among small positive integers.

## Conclusion

Session 17 is the first session to rigorously analyze the function-field analog direction.
The main finding is dual:

**Positive finding:** The function-field equation *without* the constant is completely
resolved by Mason-Stothers — only trivial solutions. This gives a clean proof of a
related (easier) statement.

**Negative finding:** The function-field equation *with* the constant $c = 5$ faces an
identical obstruction to the integer case, and for the same algebraic reason. Mason-
Stothers / polynomial-ABC is provably insufficient. The function-field version is NOT
an easier proxy for the integer version.

**Structural insight:** The constant $c = 5$ is what makes the problem hard — not the
three-term exponential structure per se. This is consistent with the cyclotomic
characterization: $c = 5$ is the unique small constant creating a $y = -3$ phantom with
a small numerator ($z = 10/3$). The hardness is intrinsic to this specific constant, not
to the general equation type.

**Implication for future strategy:** A proof of the conjecture for $c = 5$ must use:
(a) that $c = 5$ is small (size bound on the constant), and
(b) that $c = 5$ is the unique value creating a phantom with the specific arithmetic
structure $37 \cdot 27 + 1 = 10^3$, which forces $37 \mid 2^x + 5$ for all
Theorem-5 solutions.
Neither of these facts has been exploited in an essential way yet.

The core open target — an unconditional effective lower bound with exponent $\kappa < 1.107$
— remains the unique obstacle.
