## Work Done

### 1. Exact reframing as a cube-root approximation (Wolfram, symbolic)
Since `x = 5 + 332640k ≡ 2 (mod 3)`, write `x = 3a + 2`, `a = (x-2)/3 = 1 + 110880k`. Then
`2^x = 4·(2^a)^3`. With `q = 2^a`, `α = 4^(1/3) = 2^(2/3)` (`α^3 = 4`), `D = 3^y + 5`, the
equation `z^3 - 2^x = D` becomes `z^3 - α^3 q^3 = D`. Verified the **exact identity**

    z/q - α = (z^3 - α^3 q^3) / ( q^3 ((z/q)^2 + (z/q)α + α^2) ) = D/(q^3·(…)),

denominator → `3α^2 = 3·4^(2/3) ≈ 7.5595`. So `z/2^a` approximates `2^(2/3)` with exponent
`κ_actual = 3 - logD/logq`, `logD = y log3`, `logq = a log2`.

### 2. Best known measure (WebSearch)
Bennett-type effective measure: `|2^(1/3) - p/q| > 0.25 q^(-2.4325)` (best known; the ~2.43
class also covered cube roots of 5,7,11). Bennett's method covers `(b/a)^(1/n)`, so
`4^(1/3)` is in scope; the exact published constant for `4^(1/3)` was not pinned down, but
the exponent (not constant) drives the conclusion.

### 3. Contradiction criterion (Wolfram)
A solution beats the measure iff `κ_actual > κ_thm`, i.e.
`y < ((3-κ_thm)/3)(log2/log3)(x-2)`. With `κ_thm = 2.4325`:

    y < 0.119351 (x-2)   ⇔   **x > 8.3787 y**   ⟹   contradiction (no solution).

Checked across the family `(k,m)` for `k=1..4, m=0..3`: `κ_actual` ranges from negative
(when `3^y > 2^x`) up to ~2.4 only at `m=0` with growing `k`; threshold crossings computed.
Concrete sanity case `k=10, m=0`: `κ_actual = 2.762 > 2.4325` ⟹ eliminated. ✓

### 4. Sharp limitation (Wolfram)
The coefficient is `(3-κ)/3·log2/log3`. Since every irrational has irrationality exponent
≥ 2 (Dirichlet), no measure can use `κ < 2`. Best-possible threshold (`κ→2`): `x > 4.7549y`.
The balanced line `2^x=3^y` is `x = 1.585y`. So `x ≤ 4.755y` (the entire balanced and
3^y-dominant regime) is **provably unreachable by any single-cube-root approximation**.
Structural reason: `y = 3j` (Session 1) makes `3^y = (3^j)^3` an exact cube, so on the
3-side `z/3^j → 1` is rational — there is no fixed irrational to approximate. The method is
inherently one-sided.

### 5. Effect on the family & the height handle (Wolfram)
Bennett removes, for each fixed `m` (fixed `y`), all large `k`: `m=0` ⟹ `k≥5`; `m=1` ⟹
`k≥9`; `m=2` ⟹ `k≥13`; `m=3` ⟹ `k≥17`. Surviving: small `k` per `m`, infinitely many `y` —
so NOT a finish. **Key by-product:** in the surviving region `x ≤ 8.379y`,
`log z = (x/3)log2 ≤ 1.936 y`, i.e. `height(z)` is now **linearly bounded by `y`** — exactly
the lever missing in Sessions 1–2 where `z` was a free enormous base.

## Conclusion

The [HIGH] irrationality-measure direction is a **genuine but partial** tool, now precisely
quantified: Bennett's measure for `2^(2/3)` eliminates all solutions with `x > 8.379 y`, and
no measure can pass `x > 4.755 y`. The problem's hard core is therefore `x ≲ 4.75 y` (the
balanced and 3^y-dominant regime), where both exponentials are active and the natural tools
are a linear form `Λ = 3 log z - x log 2` (Baker/Matveev) or Thue-Mahler on
`z^3 - w^3 = 2^x+5`. The single most useful outcome is that, in the unresolved region,
`height(z)` is now **bounded linearly in `y`** — this dissolves the precise obstacle that
made Matveev fail in Sessions 1–2 (free, enormous `z`), so re-running Baker *restricted to
the hard core* is the clear next experiment.
