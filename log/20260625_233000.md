## Work Done

### 1. Literature watch — the two new Bugeaud papers

WebSearch surfaced (WebFetch on arxiv.org / mirrors returned HTTP 403 throughout this
session, so I worked from search-result abstracts):

- **arXiv:2604.27490** (Y. Bugeaud, Apr 2026), "On the difference between perfect powers
  and integral S-units." Establishes an **effective** lower bound for
  |z^d − q1^{a1}···qt^{at}| and for its **greatest prime factor**, tending to ∞ with z^d,
  where z is coprime to the q_i and d ≥ 2. Proof via linear forms in logarithms.
- **arXiv:2503.22084** (Y. Bugeaud, Mar 2025), "On the difference between squares and
  integral S-units." The d = 2 case: greatest prime factor of |x² − S-unit| → ∞.

These are the first 2026 results to bound exactly the Kuromine-shaped object. Two ways to
cast Kuromine as "perfect power − S-unit":

**Option (1): z³ − 2^x = 3^y + 5** (U = 2^x, S = {2}, d = 3).
- Hypothesis check (Wolfram-reasoned): z³ = 2^x + 3^y + 5; 3^y + 5 is even and 2^x is even
  (x ≥ 1), so z³ is even ⇒ z is **even** ⇒ gcd(z, 2) ≠ 1. **Bugeaud's coprimality
  hypothesis fails.** Option (1) is inadmissible.

**Option (2): z³ − 3^y = 2^x + 5** (U = 3^y, S = {3}, d = 3).
- Hypothesis check (Wolfram): for Theorem-5, x = 5 + 332640 k_x is **odd**, and
  2^x + 5 ≡ 1 (mod 3) for odd x (Table[Mod[2^x+5,3]] = 1 at odd x), so z ≢ 0 (mod 3) ⇒
  z coprime to 3. **Hypotheses satisfied.**
- The difference is exactly |z³ − 3^y| = 2^x + 5.

### 2. Quantitative assessment of Option (2) (Wolfram)

Smallest Theorem-5 candidate (k_x = k_y = 1): x = 332645, y = 332637.
- log(3^y) = 365439, log(2^x) = 230572, so **3^y dominates z³** (z³ = 2^x + 3^y + 5 ~ 3^y),
  log(z³) ≈ 365439, log z ≈ 121813.
- |z³ − 3^y| = 2^x + 5, log = 230572 = **0.631 · log(z³)**. So 2^x + 5 ≈ z^{1.89}.

A contradiction from an effective lower bound |z³ − 3^y| > z^{3−κ} requires
2^x + 5 < z^{3−κ}, i.e. 3 − κ > 230572/121813 = 1.8929, i.e. **κ < 1.107** (Wolfram).
Baker-type effective bounds — which Bugeaud's proof uses — deliver κ just below d = 3 (the
effective "gain" is ~10^−huge), giving only |z³ − 3^y| > constant. That is useless against
2^x + 5 ≈ z^{1.89}. Bugeaud's "→ ∞ with z^d" conclusion is **trivially satisfied** by
2^x + 5 → ∞, and the greatest-prime-factor version P(2^x+5) → ∞ is equally consistent. No
contradiction.

The decisive figure **0.631 = 230572/365439** is exactly the Session-3 κ_real ≈ 0.62
two-comparable-S-units barrier, now confirmed from the cube-vs-3^y side. The new paper does
not break that barrier; it sits squarely behind it.

Cross-check on neighbouring literature: arXiv:2510.04156 (arithmetic holonomy bounds,
already assessed Session 3) still yields effective irrationality measure > 2 for 2^{1/3},
far above the κ < 1.107 needed. "Multiple Legendre polynomials in Diophantine
approximation", Jacobsthal/consecutive-power 2026 papers: none unconditional in the
two-S-units regime.

### 3. Suggestion #1 — explicit 3-adic analytic map A(k_x) (Wolfram)

Goal: write k_x ↦ A as a 3-adic analytic function and decide whether A being a rational
integer at integer k_x is obstructed.

- 2^x = 32 · 2^{332640 k_x} = 32 · g^{k_x} mod 3^n, with g = 2^{332640}.
- **Wolfram:** v_3(g − 1) = 4 exactly (g ≡ 1 mod 3^4 but ≢ mod 3^5; checked n = 4..12).
  So g = 1 + 3^4 u, u a 3-adic unit.
- Hence g^{k_x} = exp_3(k_x · L) with L = log_3 g; the 3-adic log converges (v_3(g−1) = 4
  ≥ 1) and **Wolfram gives v_3(L) = 4**. Therefore
      2^x(k_x) = 32 · exp_3(k_x · L),
  and z(k_x) = (32·exp_3(k_x L) + 5)^{1/3} (the z ≡ 1 mod 3 branch — cube root of a 3-adic
  unit ≡ 37 mod 81 is analytic), and A(k_x) = z(k_x) (since 3^j → 0 3-adically), are all
  genuine **3-adic analytic functions** of k_x ∈ Z_3.

**Resolution.** By Theorem J, A(·) is a **bijection** Z_3 → Z_3. So for every integer target
a ≡ 22 (mod 27) there is a **unique** s_a ∈ Z_3 with A(s_a) = a. The question "does some
integer k_x give A(k_x) a rational integer of the right archimedean size (~2^{x/3}/3)?"
therefore reduces to "is the specific 3-adic number s_a a rational integer?" — and Z is
**dense** in Z_3, so this is **undetectable by 3-adic analysis**. There is **no
3-adic-analytic obstruction**: the explicit analytic form does not constrain integrality at
all. This confirms Session 9's expectation, now with the analytic map written down.
