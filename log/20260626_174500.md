## Work Done

### 1. The exact factorization identity and its 3-adic valuation (new: Theorem N)

For a Theorem-5 solution write $z = w + A$ with $w = 3^j$, $A = z - 3^j \geq 1$. The
Theorem-5 equation $z^3 = 2^x + 3^{3j} + 5$ gives $2^x + 5 = z^3 - w^3$. Subtracting
$A^3$ from both sides yields the **identity** (verified symbolically in Wolfram):

$$2^x + 5 - A^3 = (w+A)^3 - w^3 - A^3 = 3w^2 A + 3w A^2 = 3\,w\,A\,(w + A) = 3^{j+1}\,A\,(A + 3^j).$$

By **Theorem I**, $A \equiv 22 \pmod{27}$, so $\gcd(A, 3) = 1$. Since $j \geq 1$,
$A + 3^j \equiv A \not\equiv 0 \pmod 3$, hence $v_3\!\big(A(A+3^j)\big) = 0$ and

$$\boxed{\,v_3\!\big(2^x + 5 - A^3\big) = j + 1\,}\qquad\text{(Theorem N).}$$

**Verification (Wolfram):**
- Known $(x,A,j) = (5,1,1)$: $2^5 + 5 - 1 = 36 = 3^{2}\cdot 4 = 3^{j+1}A(A+3^j)$, $v_3 = 2 = j+1$. ✓
- Known $(x,A,j) = (1,1,0)$: $2^1 + 5 - 1 = 6$, $v_3 = 1 = j+1$. ✓

**LTE corollary for the $A = 1$ band.** Setting $A = 1$ gives
$2^x + 4 = 3^{j+1} + 3^{2j+1}$, and (for $x$ odd, true throughout Theorem-5)
$$v_3(2^x + 4) = v_3\!\big(4(2^{x-2}+1)\big) = v_3(2^{x-2}+1) = 1 + v_3(x - 2)$$
by Lifting-the-Exponent. For Theorem-5, $x - 2 = 3 + 332640\,k_x = 3\,(1 + 110880\,k_x)$
and $110880 \equiv 0 \pmod 3$, so $1 + 110880\,k_x \equiv 1 \pmod 3$, giving
$v_3(x-2) = 1$ and hence $v_3(2^x + 4) = 2$ **universally** (Wolfram: constant $2$ for all
$k_x \in [0,5]$). By Theorem N the $A = 1$ band requires $j + 1 = 2$, i.e. $j = 1$ — which
is exactly the known solution $(5,3,4)$. So the $A = 1$ band contains *no* Theorem-5
solution (all have $j \geq 55439$). This is an **independent valuation-theoretic proof** of
the Session-4/5 fact "$A \neq 1$ for Theorem-5," not using any size estimate.

More generally, for any fixed integer $a$ coprime to 3, the valuation
$v_3(2^x + 5 - a^3)$ is a **small bounded constant** across the whole Theorem-5 family
(Wolfram: $a = 1,2,4,5,7,8 \Rightarrow v_3 = 2,0,3,0,2,0$, constant over $k_x = 0..3$),
whereas Theorem N requires it to equal $j+1 \geq 55440$. Hence **no fixed small $a$** is a
possible value of $A$ — re-deriving cleanly, via 3-adic valuation, that $A$ must be huge.

### 2. Is $A$ actually forced to be small for surviving cases? (The decisive test)

The coprime factorization $A\cdot B = 2^x + 5$ with $B = z^2 + z\,3^j + 3^{2j} > 3^{2j}$
gives the **upper bound**
$$A < \frac{2^x + 5}{3^{2j}} \approx 3\,e^{D},\qquad D := x\log 2 - (2j+1)\log 3.$$
$A$ is genuinely small (say $O(1)$) **only** when $D \approx 0$, i.e. exactly at the
Theorem-M boundary $2^x \approx 3^{2j+1}$. (Theorem M eliminates $D < 0$; the surviving
region is $D \geq 0$.)

**The question reduces to:** how small can $D \geq 0$ be over the Theorem-5 lattice
$x = 5 + 332640\,k_x$, $2j+1 = 110879 + 110880\,k_y$? This is an inhomogeneous Diophantine
approximation:
$$D = c_0 + k_x\,P - k_y\,Q,\quad P = 332640\log 2,\ Q = 110880\log 3,\ \tfrac{P}{Q} = \tfrac{3\log 2}{\log 3} \approx 1.8928.$$

**Wolfram computation (over $k_x \leq 5000$, choosing optimal $k_y$ per $k_x$):**
- Overall closest approach: $|D| \approx 9.96$ at $(k_x, k_y) = (1054, 1994)$ — but this
  has $D < 0$ (it is *eliminated* by Theorem M).
- Closest **surviving** approach ($D \geq 0$): $D \approx 54.99$ at $(k_x, k_y) =
  (4729, 8950)$, giving $A \gtrsim 3\,e^{55} \approx 6.3 \times 10^{24}$.
- For $k_x \leq 200$, the closest surviving $D$ values are in the hundreds-to-thousands;
  e.g. $(28, 52)$ gives $|D| \approx 227$ (and that one is on the eliminated side).

**Conclusion of the test:** Over the entire feasible search range, **no surviving
Theorem-5 pair ever has $A$ smaller than $\sim 10^{24}$**, and almost all surviving pairs
have $A$ astronomically large ($\log_3 A \geq 87000$ for the first several $k_x$). The
bounded-$A$ elimination — which would require $A$ to be a small fixed integer so that
Theorem N's valuation contradiction $j+1 \leq v_3(2^x+5-a^3) = O(1)$ fires — **cannot
trigger for the Theorem-5 family.**

### 3. Why: the lattice never lands near the boundary (structural diagnosis)

The reason is the same coarseness that blocks every congruence approach, now seen
metrically: the family steps by $P \approx 230575$ in $k_x$ and $Q \approx 121814$ in
$k_y$ (in log units). The boundary $D = 0$ is a line of irrational slope $1.8928$ through
the integer lattice $(k_x, k_y)$; the family lands on lattice points whose distance to
this line is governed by $\{1.8928\,k_x\}$ and is generically of order $1$ in the
$(k_x,k_y)$ grid — but each unit of grid distance is $\sim 10^5$ in $D$, hence
$A \sim e^{D}$ is never $O(1)$. The known solutions live at $k_x = 0$-analog,
$D = 5\log2 - 7\log3 \cdot(\ldots)$ effectively $= 0$ exactly only because they are the
genuine small solutions outside the $k_x \geq 1$ progression.

This is the metric face of the two-comparable-S-units barrier: the family is *near* the
$2^x = 3^y$ boundary in *exponent ratio* (slope $1.8928$) but never *on* it to within
$O(1)$ in absolute log-distance. So $A$, which measures exactly that absolute distance, is
never small.

## Results

1. **Theorem N (new, exact, elementary):** For every Theorem-5 solution,
   $$2^x + 5 - A^3 = 3^{j+1}\,A\,(A + 3^j),\qquad v_3(2^x + 5 - A^3) = j + 1,$$
   where $A = z - 3^j$ and $\gcd(A,3) = 1$ (by Theorem I). Verified on both known
   solutions. This is a clean exact valuation identity.

2. **Valuation re-proof that $A$ is huge:** For any fixed $a$ coprime to 3,
   $v_3(2^x + 5 - a^3)$ is a small constant over the family, while Theorem N demands it
   equal $j + 1 \geq 55440$. Hence $A$ cannot be any fixed small integer — an independent,
   purely 3-adic proof of the Session-4 size fact. The $A = 1$ band is eliminated by LTE:
   $v_3(2^x + 4) = 2$ universally, forcing $j = 1$ (only the known solution).

3. **Negative result (the main finding):** The bounded-$A$ elimination strategy **cannot
   work** for the Theorem-5 family. Over $k_x \leq 5000$, no surviving pair has
   $A \lesssim 10^{24}$; the family never approaches the Theorem-M boundary $2^x = 3^{2j+1}$
   closely enough (closest surviving $D \approx 55$, closest overall $D \approx 10$ but on
   the eliminated side). $A$ is small only in a measure-zero neighborhood of the boundary
   that the integer lattice misses.

4. **Structural clarification:** Theorem N is a tautological reformulation for a genuine
   solution (it is automatically satisfied), so it adds *no* constraint on its own; its
   value is the clean valuation formula and the re-proof of "$A$ huge." The hoped-for
   contradiction (bounded $A$ + small valuation) requires the family to sit near the
   boundary, which it provably does not. This is the metric/Diophantine-approximation face
   of the two-comparable-S-units barrier.

## Conclusion

Session 15 fully resolves the Session-14 bounded-$A$ exploration, **negatively but
cleanly**. The new exact valuation identity (Theorem N) is correct and worth recording,
and it gives an elegant independent proof that $A$ cannot be any small fixed integer.
However, the strategy of forcing a contradiction from "small $A$ + bounded 3-adic
valuation" is provably blocked: the Theorem-5 lattice never lands within $O(1)$
log-distance of the Theorem-M boundary, so $A$ is never small enough (always
$\gtrsim 10^{24}$, usually astronomically larger).

This is the same two-comparable-S-units barrier, now seen through a third lens (after the
analytic/irrationality-measure lens of Session 3 and the congruence lens of Sessions 4–9):
metrically, the family is near the $2^x = 3^y$ boundary in *slope* but never in *absolute
distance*. Every attempt to exploit closeness to the boundary fails for the same reason an
effective lower bound with exponent $\kappa < 1.107$ is needed and unavailable.

The core open target is unchanged.
