### 1. Soft-obstruction re-verification (Wolfram)

The phantom is $(x,y,z)=(5,-3,10/3)$. Its RHS:
$$2^5 + 3^{-3} + 5 = 32 + \tfrac{1}{27} + 5 = \tfrac{1000}{27} = \left(\tfrac{10}{3}\right)^3.$$
Wolfram confirmed $2^5+3^{-3}+5 = 1000/27 = (10/3)^3$ exactly.

For the Theorem-5 sieve, a candidate prime $p$ with $\gcd(p,3)=1$ constrains $x\bmod
\mathrm{ord}_p(2)$ and $y\bmod\mathrm{ord}_p(3)$. Along the phantom residues
($x\equiv5$, $3^y\equiv3^{-3}\pmod p$) the RHS reduces to $1000/27 \pmod p$. Since $1000/27$
is a **global rational cube**, it is a cube in $\mathbb{Q}_p$ for every $p\neq3$, so the
sieve can never exclude these residues.

**Wolfram test:** over all 195 primes $p\equiv1\pmod3$ with $p \le \mathrm{Prime}(400)\approx
2741$ (the hardest case, since cubes are only $(p-1)/3$ of the residues when $p\equiv1\pmod3$):
$1000\cdot27^{-1}\bmod p$ is a cube residue in **195 / 195** cases. Confirmed soft obstruction.

*Self-correction recorded for honesty:* my first run mistakenly tested $37/27$ (i.e.
$(2^5+5)\cdot27^{-1}$, omitting the $+3^{-3}$ that makes the numerator $1000$, not $999$),
which is **not** the phantom RHS; it gave only 63/195 cube residues. Using the correct
phantom value $1000/27$ restores 195/195. The dead-end record is correct; the transient error
was mine, not the literature's.
