## Work Done

### 1. Irrationality-measure reframing and quantitative test (Wolfram)
With $x=3a+2$ (forced since $x\equiv5\equiv2\pmod3$), $2^x=4\cdot8^a=4W^3$, $W=2^a$. So
$z^3-4W^3=3^y+5$, a cubic Thue form; dividing by $W^3$, $z/W$ approximates
$\alpha=4^{1/3}=2^{2/3}$, with $|\alpha-z/W|\approx(3^y+5)/(3\alpha^2W^3)$.

Computed at the minimum case $x=332645$, $y=166317$, $a=110881$:
- $\log W = a\log2\approx76856.9$
- $\log(3^y+5)\approx y\log3\approx182718$
- $\log|\alpha-z/W|\approx-47854.7$
- **Realized exponent $\kappa_{\text{real}}=-\log|\alpha-z/W|/\log W\approx0.6226$.**

### 2. Two independent proofs the method fails
- **Quality test:** $\kappa_{\text{real}}\approx0.62$ is below any effective $\mu\ge2$
  (Baker $2.96$, Chudnovsky/Bennett $2.43$, holonomy arXiv:2510.04156 sharper but $>2$).
  An irrationality measure is a *lower* bound, useful only on *good* approximations
  ($\kappa\ge\mu$). $z/W$ is *bad*, so no measure constrains it.
- **Thue-value test:** contradiction needs $\log(3^y+5)<(3-\mu)\log\max(z,W)$. With
  $\mu=2.43$: RHS $\approx43809$, LHS $\approx182718$, gap $+138909>0$. No contradiction.

**Root cause identified:** $3^y$ is not a small perturbation — $\log3^y/\log2^a\approx2.38$,
i.e. $3^y\approx z^2$. The equation is in the "two comparable dominant S-units" regime;
no single cubic irrationality measure (any $\mu$) can resolve it. Re-confirmed the Matveev
3-log bound is too weak by $\sim10^{10}$ (constant $\approx4.1\times10^{13}$, bound
$\approx-5.6\times10^{14}$ vs actual $-47854$).

### 3. Literature check (WebSearch)
Confirmed arXiv:2510.04156 (Calegari–Dimitrov–Tang, Oct 2025) gives effective
irrationality measures for high-order roots of an algebraic number via arithmetic
holonomy bounds — exactly the $|2^{1/3}-p/q|$ tool. It sharpens $\mu$ but, per §2, cannot
help when $\kappa_{\text{real}}\approx0.62$.

### 4. Algebraic factorization probe (Wolfram)
- Derived a clean **equivalent single equation**. With $A=z-3^j$, $w=3^j$:
  $$2^x+5 = A^3+3^{j+1}A^2+3^{2j+1}A = A\,(A^2+3^{j+1}A+3^{2j+1}).$$
  Brute force over $x\le40$ (all $j$) recovers exactly the two known solutions:
  $(x,j,A,z)=(1,0,1,2)$ and $(5,1,1,4)$ — **both with $A=z-3^{y/3}=1$.**
- **Eisenstein-norm identity:** $4B-3w^2=(2z+w)^2$ where $B=z^2+zw+w^2$. Verified on
  $(5,3,4)$: $4\cdot37-3\cdot9=121=11^2$.
- **Negative result:** the inert-prime divisibility ($q\equiv2\pmod3$, $q\mid2^x+5
  \Rightarrow q\mid z-3^j$) is *automatically* consistent: combined with the equation it
  gives $3^{3j}\equiv3^y\pmod q$, automatic because $y=3j$. No new congruence — another
  manifestation of the soft-obstruction principle.

## Conclusion

The board of *effective* size-based methods is now cleared: local methods (Session 2),
single irrationality measures / 2-term linear forms (this session), and full 3-log Baker
(Session 1) are all either provably insufficient or quantitatively too weak by enormous
margins, all for the same underlying reason — two comparable large exponential terms and
an obstructing global rational cube. The honest situation is that the Kuromine Problem
sits beyond all currently known effective techniques for this parameter range.

The most promising *non-size* lead remaining is the algebraic factorization, now in the
sharp form $2^x+5=A(A^2+3^{j+1}A+3^{2j+1})$ with $B=A^2+3^{j+1}A+3^{2j+1}$ an Eisenstein
norm all of whose primes are $\equiv1\pmod3$. The avenue worth pursuing is a
growth/primitive-divisor argument (Zsygmondy / Bilu–Hanrot–Voutier) on $B$ along the
$x$-progression — a tool that uses the integrality of $w=3^j$ (and thus can escape the
soft obstruction) rather than raw size.
