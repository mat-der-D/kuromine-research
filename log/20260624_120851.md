## Results

- **NEW THEOREM (unconditional):** Any solution with x ≥ 6, y ≥ 2 satisfies
  2^x + 5 ≥ 3^{2y/3 + 1}, equivalently x > (2/3)·log₂(3)·y ≈ 1.0566·y.
  This rules out the entire half-plane x ≲ 1.0566 y. (Surviving region is a one-sided cone.)

- **d = 1 stratum fully resolved:** 3m² − 19 = 2ⁿ has only (n,m) = (3,3) and (7,7) for n ≤ 400.
  (3,3) → y = 0 (outside y ≥ 2); (7,7) → the known solution (5,3,4). So no new solutions have
  d = 1. The known large solution lives exactly here. Rigorous finiteness for all n should
  follow from standard Ramanujan–Nagell / Baker techniques (to be cited or proved next).

- **Empirical:** Over all d ≤ 500 and y ≤ 240, the only (d, y, x) giving a power of 2 is the
  known solution. No solution with x, y ≤ 60 beyond the two known.

- **3-adic clarification (negative-ish):** 37 is a true ℤ₃-cube, so the naive hope of using
  "37 is not a 3-adic cube" to obstruct is void — consistent with the dead-end note. The
  phantom (10/3) is genuinely a different object (non-integral 3-adically).

## Conclusion

The Theorem-5 corollary y ≡ 0 (mod 3) is more powerful than previously used: it turns the
hard remaining case into an S-unit / Ramanujan–Nagell problem via an elementary factorization.
We now have (a) a clean unconditional lower bound x > 1.0566 y, and (b) a complete kill of the
d = 1 stratum. The problem is reduced to: bound d (equivalently bound x above in terms of y)
and dispose of d ≥ 2. This is squarely in Baker's-method territory, so the previously
open-ended Baker direction is now concrete and actionable.

Honest caveats: the size theorem alone does NOT close the problem (it leaves an infinite cone);
the "only n ∈ {3,7}" claim for 3m²−19=2ⁿ is verified to n ≤ 400 but a full proof of finiteness
is deferred; the d ≥ 2 cases are only empirically checked so far, not proved impossible.
